## MERN Stack – Core Understanding

1. What is the MERN stack? Explain each component’s role.

The **MERN stack** is ==a popular, open-source JavaScript framework used for building robust, full-stack web applications==. The acronym stands for MongoDB, Express, React, and Node.js, allowing developers to use a single language, JavaScript, across the entire development process

1. Why is JavaScript used across the entire MERN stack?

JavaScript is used across the entire MERN stack

==because all its components—**MongoDB, Express.js, React, and Node.js**—either use or are built upon JavaScript, providing a unified language ecosystem for the entire development process==
